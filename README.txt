== Description ==

WordPress plugin to allow readers to comment the review itself. 

Activating plugin generates a custom post type and includes simple settings page. 

From settings page it is possible to display score or display reviews based on category.

EXAMPLE usage.

Create a text widget for a footer.
Write "[add-review]" shortcode.
Write a normal WordPress post and you should see a form to write a review about the post

== Installation ==


1. Upload `reviews.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
